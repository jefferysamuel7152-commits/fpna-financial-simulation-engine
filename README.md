# Financial Simulation & FP&A Engine

## Overview
Built a multi-scenario FP&A simulation engine using Python and Pandas to analyze margin sensitivity under cost fluctuations.

## Features
- Profit & Margin Calculation
- Scenario Simulation (0%, 10%, 20% cost increase)
- Region & Product Level Analysis
- Decision Engine (Risk Classification)
- Visualization (Matplotlib)

## Key Insights
- Identified margin compression under cost inflation
- Detected high-risk product (Product D)
- Provided strategic recommendations for cost control and pricing

## Tools Used
- Python
- Pandas
- Matplotlib

## Output
Includes margin sensitivity chart and dataset-driven analysis.
